# Huma-volve-huma-volve-round8-backend-team-one

Welcome to the project! This README will guide you through the basic Git workflow for working on this repository, including creating branches, making changes, and creating Pull Requests.

---

## 1. Clone the Repository

If you haven't downloaded the project yet, clone it from GitHub:

```bash
  git clone https://github.com/Huma-volve/Huma-volve-huma-volve-round8-backend-team-one.git  
```
## 2. Creating a New Branch

Make sure you are on the main branch:

 git checkout main  

Create a new branch and switch to it:
```bash
 git checkout -b your-branch-name 
```
## 3. Making Changes

Edit or add files in the project.

Check the status of your changes:
```bash
 git status 
```

Stage your changes:
```bash
 git add .  
```

Commit your changes:
```bash
 git commit -m "Add a brief description of your changes" 
```
## 4. Pushing Your Branch to GitHub

Push your new branch to the remote repository:
```bash
 git push origin branch-name
```
